Sylvite is a Kinesis Client Library (KCL) multi-lang deamon runner. The multi-lang deamon allows non-Java code to process a Kinesis stream, while still benefiting from the checkpointing, resharding, load balancing and monitoring capabilities provided by the KCL.

Sylvite is a lightweight wrapper around the KCL which makes it easier to package and deploy multi-lang processors. One could say it feels much more natural than the KCL.
